# 坦克大战

经典坦克大战网页游戏，使用纯 HTML5 + Canvas 实现，无需任何依赖，打开浏览器即可游玩。

## 在线游玩

通过 GitHub Pages 即可在浏览器中直接体验：

1. 进入仓库 **Settings → Pages**
2. Source 选择 `main` 分支
3. 保存后访问 `https://xyyhx.github.io/tank_battle/`

## 操作说明

| 按键 | 功能 |
|------|------|
| W / ↑ | 向上移动 |
| S / ↓ | 向下移动 |
| A / ← | 向左移动 |
| D / → | 向右移动 |
| 空格 / J | 射击 |
| P | 暂停 |

## 游戏特性

- 🎮 玩家坦克控制，WASD / 方向键移动，空格射击
- 🤖 敌方坦克 AI，随关卡增强
- 🧱 可破坏的砖墙
- 🛡️ 护盾道具，拾取后短暂无敌
- 💥 爆炸特效与子弹碰撞
- 📈 关卡递进，敌人数量与速度逐步提升
- ❤️ 3 条生命，被击中后重生带短暂护盾

## 技术实现

- 纯前端单文件实现（`index.html`），零依赖
- HTML5 Canvas 2D 渲染
- `requestAnimationFrame` 游戏主循环
- 基于瓦片的地图系统
- 简易碰撞检测与 AI 寻路
